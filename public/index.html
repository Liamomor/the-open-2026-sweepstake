How it works

  1. Each entrant has three golfers.
  2. All three must make the cut.
  3. The three live scores are added together.
  4. Lowest combined score after Sunday wins.

Cut and withdrawal

Any team with a cut, withdrawal or disqualification is clearly marked and moved below valid teams.

Data mode

The site starts with demo scores. The organiser can paste real scores using the settings button, so the whole competition can run even without a paid API.
