The Open 2026 Live Display — Easy Setup

This package already contains all 341 competition entries. You only need to host it and add the live-score key.
  1. Create free accounts with GitHub, Cloudflare, and Slash Golf / Live Golf Data on RapidAPI.
  2. On GitHub, create a repository called the-open-2026-sweepstake.
  3. Upload the full contents of this downloaded folder, including the public and functions folders.
  4. In Cloudflare choose Workers & Pages → Create → Pages → Connect to Git, then select your repository.
  5. Use no build command. Set the output directory to public.
  6. After it deploys, add these Variables and Secrets in Cloudflare:
    • RAPIDAPI_KEY — your private API key
    • TOURN_ID — The Open's tournament ID
    • YEAR — 2026
    • ORG_ID — 1
  7. Redeploy and open your new .pages.dev address.
  8. Send that address to the group. Everyone sees the same live standings.

Never paste your API key into WhatsApp or into the public website files.
